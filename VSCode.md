# VS Code

Notes related to exploring and adopting the VS Code tool as a replacement for the IntelliJ suite.

```
brew install --cask visual-studio-code
```

## Hot Keys

| Keybinding | Action |
| --- | --- |
| `⌘` `,` | Open Settings |
| `⌘` `o` | Open Folder |
| `⌘` `p` | Go to File (or line, or symbol`) |
| `⌘` `⇧` `P` | Open Command Palette |
| `⌘` `k` `⌘` `s` | Edit Keyboard Shortcuts |
| `⌃` `` ` `` | Open Terminal |
| `⌘` `n` | New File |
| `⌘` `⇧` `N` | New Code Workspace |
| `⌘` `f` | Find in File |
| `⌘` `⌥` `f` | Find and Replace in File |
| `⌘` `⇧` `F` | Find in Project |
| `⌘` `⇧` `H` | Replace in Project |
| `⌘` `⌥` `↓` | Multiline Cursor |


## Palette Commands
